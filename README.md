# BApparels Online Store

Welcome to the BApparels Online Store! This is a simple e-commerce web application built with React and Redux. It allows users to browse and purchase a variety of products.

## Features

- View a list of available products
- View detailed information about each product
- Add products to the shopping cart
- Adjust the quantity of products in the cart
- Remove products from the cart
- Proceed to checkout and provide shipping information
- Empty the cart.

## Installation

1. Clone the repository:

```

git clone <https://github.com/Princebounce/BApparels/>


```

2. Install the dependencies:

```
npm install

```

3. Start the development server:

```
npm start

```

4. Open your web browser and visit `http://localhost:3000` or corresponding IP address on same network with localhost to view the application.

## Technologies Used

- React
- Redux
- React Router
- Axios

## Project Structure

The project has the following structure:

- **src/components**: Contains the main components of the application, such as Home, Products, Product, Cart, Checkout, etc.
- **src/redux**: Contains the Redux actions, reducers, and store configuration.
